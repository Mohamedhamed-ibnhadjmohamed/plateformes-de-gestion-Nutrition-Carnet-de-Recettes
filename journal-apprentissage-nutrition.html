Objectif du projet

ASP.NET Core MVC Entity Framework Core SQL Server CRUD + Dashboard

3 tables, 1 relation N-N
Ingredient PK
ChampTypeDescription
Idint🔑 Clé primaire auto-incrémentée
NomstringNom de l'ingrédient (ex: Riz)
CaloriesParUnitedoubleCalories pour 1 unité (gramme)
Recette PK
ChampTypeDescription
Idint🔑 Clé primaire
NomstringNom de la recette
NbPersonnesintNombre de personnes
CategoriestringPetit-déjeuner / Déjeuner / Dîner / Dessert
TypeCuisinestringTunisienne / Française / Italienne / Méditerranéenne
RecetteIngredient PK FK → Recette FK → Ingredient
ChampTypeDescription
Idint🔑 Clé primaire
RecetteIdint🔗 FK vers Recette.Id
IngredientIdint🔗 FK vers Ingredient.Id
QuantitedoubleQuantité en grammes
Relation N-N : Une recette contient plusieurs ingrédients, et un ingrédient peut apparaître dans plusieurs recettes. La table RecetteIngredient est la table de jointure.

Modèles C#

Les 3 classes du dossier Models/
C# — Ingredient.cs // Models/Ingredient.cs public class Ingredient { public int Id { get; set; } [Required] public string Nom { get; set; } public double CaloriesParUnite { get; set; } // Navigation property public ICollection<RecetteIngredient> RecetteIngredients { get; set; } }
C# — Recette.cs // Models/Recette.cs public class Recette { public int Id { get; set; } [Required] public string Nom { get; set; } public int NbPersonnes { get; set; } public string Categorie { get; set; } // "Petit-déjeuner" | "Déjeuner" | "Dîner" | "Dessert" public string TypeCuisine { get; set; } // "Tunisienne" | "Française" | "Italienne" | "Méditerranéenne" public ICollection<RecetteIngredient> RecetteIngredients { get; set; } }
C# — RecetteIngredient.cs // Models/RecetteIngredient.cs public class RecetteIngredient { public int Id { get; set; } public int RecetteId { get; set; } public Recette Recette { get; set; } public int IngredientId { get; set; } public Ingredient Ingredient { get; set; } public double Quantite { get; set; } }

Module 4

DbContext & EF Core

Configuration de la base de données
C# — ApplicationDbContext.cs // Data/ApplicationDbContext.cs public class ApplicationDbContext : DbContext { public ApplicationDbContext(DbContextOptions<ApplicationDbContext> options) : base(options) { } public DbSet<Ingredient> Ingredients { get; set; } public DbSet<Recette> Recettes { get; set; } public DbSet<RecetteIngredient> RecetteIngredients { get; set; } protected override void OnModelCreating(ModelBuilder modelBuilder) { // Relation Recette → RecetteIngredient modelBuilder.Entity<RecetteIngredient>() .HasOne(ri => ri.Recette) .WithMany(r => r.RecetteIngredients) .HasForeignKey(ri => ri.RecetteId); // Relation Ingredient → RecetteIngredient modelBuilder.Entity<RecetteIngredient>() .HasOne(ri => ri.Ingredient) .WithMany(i => i.RecetteIngredients) .HasForeignKey(ri => ri.IngredientId); } }
C# — Program.cs (extrait) // Program.cs builder.Services.AddDbContext<ApplicationDbContext>(options => options.UseSqlServer( builder.Configuration.GetConnectionString("DefaultConnection") ) );
Bash — Migrations EF Core # Créer la migration initiale dotnet ef migrations add InitialCreate # Appliquer en base de données dotnet ef database update

Module 5

Indicateurs & LINQ

Les calculs clés pour le dashboard (important pour la note !)
Formule 1 — Calories totales
CaloriesTotales = Σ ( Quantité × CaloriesParUnité )
Exemple : Recette Couscous
IngrédientQuantité (g)Cal/unitéTotal kcal
Semoule2001.3260
Viande3002.0600
TOTAL——860 kcal
Formule 2 — Calories par personne
CaloriesParPersonne = CaloriesTotales ÷ NbPersonnes
Ex : 860 ÷ 4 = 215 kcal/pers.
C# LINQ — DashboardController.cs // 1. Calories totales d'une recette var caloriesTotales = _context.RecetteIngredients .Where(ri => ri.RecetteId == id) .Sum(ri => ri.Quantite * ri.Ingredient.CaloriesParUnite); // 2. Calories par personne var recette = _context.Recettes.Find(id); var calParPersonne = caloriesTotales / recette.NbPersonnes; // 3. Répartition par catégorie var parCategorie = _context.Recettes .GroupBy(r => r.Categorie) .Select(g => new { Categorie = g.Key, Nombre = g.Count() }).ToList(); // 4. Analyse par type de cuisine var parCuisine = _context.Recettes .GroupBy(r => r.TypeCuisine) .Select(g => new { TypeCuisine = g.Key, Nombre = g.Count() }).ToList();
